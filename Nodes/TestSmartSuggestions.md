# Тест умных подсказок

## Сценарий 1: Юмор без точного триггера
Запрос: "Мне скучно, развей меня"

Ожидаемая реакция:
Ассистент: "Кстати, у меня есть нода с анекдотами! Хочешь, чтобы я рассказал что-нибудь смешное?"

## Сценарий 2: Здоровье с синонимами  
Запрос: "Как моё самочувствие сегодня?"

Ожидаемая реакция:
Ассистент: "У меня есть нода для проверки здоровья системы! Хочешь запустить ежедневный чек?"

## Сценарий 3: Финансы (строгий режим)
Запрос: "Посчитай мои деньги"

Ожидаемая реакция: 
Без подсказки (строгий режим для финансовых тем)
