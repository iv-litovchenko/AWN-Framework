# Automation Manager

> [!info] О файле  
> Системная нода для управления автоматизацией: создаёт крон-задачи на основе нодов с `AWN-AUTOMATIZATION: true`.

## Принцип работы

1. **Сканирование**: каждые 10 минут проверяет все ноды в реестре
2. **Фильтрация**: ищет ноды с `AWN-AUTOMATIZATION: true`
3. **Создание задач**: для каждой такой ноды создаёт крон-задачу через навык `cron`
4. **Мониторинг**: отслеживает выполнение и логирует результаты

## Формат крон-задач

Каждая задача создаётся с уникальным ID на основе пути к ноде:
- ID задачи: `automation_{md5_пути_к_ноде}`
- Команда: вызов агента с указанием ноды для выполнения

## Логирование

Все результаты выполнения сохраняются во внешней памяти:
`Automation/AutomationLog.memory.md`

## Пример использования

Создайте ноду с:
- `AWN-AUTOMATIZATION: true`
- `AWN-CRON: "расписание"`
- Чёткими инструкциями в теле ноды

Эта нода автоматически подхватится и получит свою крон-задачу.
