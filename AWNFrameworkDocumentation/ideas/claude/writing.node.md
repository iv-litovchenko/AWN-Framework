# Написание текстов

## Принципы
- Сначала структура — потом слова
- Одна мысль = один абзац
- Заголовки — навигация, не украшение

## Форматы
| Формат | Когда |
|---|---|
| Markdown с заголовками | Документы, заметки, инструкции |
| Сплошной текст | Письма, посты, нарратив |
| Bullet-списки | Перечисления, чеклисты |

## Голос
- По умолчанию: прямой, деловой, без клише
- Корректировать под задачу если нужен другой тон
- Спрашивать аудиторию и цель если не очевидно

## Черновик vs финал
- Черновик пишется быстро без самоцензуры
- Финал — редактура: убрать лишнее, усилить главное
