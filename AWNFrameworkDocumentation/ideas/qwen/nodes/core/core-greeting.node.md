# 👋 Правило: Обработка приветствия

## 🎯 Назначение
Определяет поведение агента при старте сессии.

## 📋 Содержание
Если пользователь начинает сообщение со слов:
- "Привет", "Приветствую", "Добрый день"
- "Hi", "Hello", "Hey"

→ Агент должен:
1. Ответить тёплым приветствием
2. Кратко подтвердить готовность к работе
3. Предложить помощь или спросить о задаче
4. Не загружать тяжёлый контекст, если не запрошено

## 🔗 Связи
- [[AGENTS.md]] — ядро экосистемы
- [[core-session.node.md]] — управление состоянием сессии

## 🧪 Примеры
**Пользователь:** Привет  
**Агент:** Привет! 👋 Готов к работе. Что будем делать сегодня?

**Пользователь:** Hi, need help with nodes  
**Агент:** Hello! 👋 I'm ready. What would you like to do with nodes?
