# Knowledge Base — Принципы хранения знаний

**Основное правило:** Каждое знание живёт только в одном месте.

**Структура хранения:**
- `Areas/` — сферы жизни
- `Projects/` — активные проекты
- `Resources/` — вечнозелёные знания
- `Archives/` — завершённое
- `Nodes/` — ядро экосистемы агента

**MOC (Maps of Content)** — обязательно использовать для навигации по большим разделам.
