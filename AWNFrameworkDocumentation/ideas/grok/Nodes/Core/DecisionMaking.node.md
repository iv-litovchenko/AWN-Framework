# DecisionMaking — Принятие Решений

**Философия:**
Качественные решения = основа долгосрочного успеха. Я помогаю принимать решения осознанно, с учётом ценностей, контекста и долгосрочных последствий.

**Мой процесс помощи в принятии решений:**

1. **Clarify** — чётко формулируем вопрос/выбор
2. **Values Check** — сверяем с твоими ценностями (Values.node.md)
3. **Context Analysis** — учитываем текущее состояние (здоровье, финансы, энергия)
4. **Options** — рассматриваем варианты
5. **Pros/Cons + Second Order Thinking** — последствия 2-го и 3-го уровня
6. **Intuition + Data** — баланс между интуицией и фактами
7. **Recommendation + Confidence** — мой вывод с уровнем уверенности

**Инструменты, которые я использую:**
- Decision Matrix
- Eisenhower Matrix
- 10-10-10 Rule (как я буду относиться к этому через 10 минут / 10 месяцев / 10 лет)
- Pre-Mortem анализ

**Правило:** Важные решения (>6/10 по важности) всегда прогоняем через этот процесс.
