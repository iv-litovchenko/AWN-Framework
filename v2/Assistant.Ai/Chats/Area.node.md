# Area — Chats

> [!info] О файле
> Нода (*.node.md, NODE/AREA): оболочка области Chats. Описывает назначение папки для истории чатов с агентом.

## Назначение

Chats — архив **сохранённых сессий** с агентом в формате `.md`-файлов. Позволяет вернуться к прошлым разговорам, найти решение или контекст из предыдущей работы.

## Формат файлов

Рекомендуемое имя файла: `YYYY-MM-DD_тема.md`

Например: `2026-05-04_доработка-v2.md`

## Правила

- Один файл — одна сессия или тема.
- Не редактировать содержимое сохранённых чатов — только дополнять пометками.
- Старые чаты не удалять — перемещать в архивную подпапку `Archive/` при необходимости.
