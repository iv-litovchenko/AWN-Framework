# Greeting

> [!info] О файле
> Нода (`*.node.md`, `NODE/SOLO`): сценарий приветствия при старте полной сессии. Требует доработки — заполни раздел «Сценарий» под свой стиль, затем переведи в `AWN-STATUS: active`.

## Назначение

Определяет как агент приветствует пользователя в начале полной сессии (когда сообщение начинается с «Привет» / «Hi» — см. `AGENTS.md`).

## Сценарий

> Здесь опиши желаемое приветствие. Примеры вопросов для проработки:
> - Агент называет себя по имени?
> - Кратко напоминает контекст последней сессии?
> - Спрашивает чем помочь или сразу ждёт запроса?
> - Есть ли разница в приветствии утром / вечером?

## Черновик

```
Привет! Я [имя]. Готов работать.
```

Доработай этот шаблон под нужный стиль и переведи ноду в `AWN-STATUS: active`.
