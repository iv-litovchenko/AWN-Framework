## Назначение

Core — папка для **системных скриптов и инструментов**, которые обеспечивают работу ассистента: автоматизации, вспомогательные утилиты, технические заготовки.

Сейчас папка является **заглушкой** — наполняется по мере появления реальных скриптов.

## Что может здесь лежать

- Скрипты автоматизации (cron, запуски)
- Вспомогательные утилиты для работы с vault
- Технические конфиги и пресеты
