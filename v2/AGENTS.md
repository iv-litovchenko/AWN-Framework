# AGENTS.md

Минимальная точка входа протокола AWN v2. Определяет **режим сессии**; полный текст протокола (ноды, принципы, команды) — в `./AGENTS.FULL.md`.

> **Где мы находимся:** текущая папка — **Vault** (хранилище Obsidian). Он же **workspace агента и пользователя**. Не выходи за пределы без явного разрешения пользователя.

> **Карта нод** — в `./AGENTS.NODES.md`. Настройки агента — `./AGENTS.SETTINGS.md`.

> **Среды с автоинжекцией** (например Cursor: workspace rules): этот файл может попадать в контекст всегда. Имеет значение **явное чтение** `./AGENTS.FULL.md`: в **чистой** сессии его **не** открывать — так экономится объём полного протокола.

---

## Bootstrap (обязательно)

Перед первым ответом определи режим по **первому сообщению пользователя** в этой беседе.

**Полная сессия** — если первая реплика пользователя **начинается с приветствия** **«Привет»** или **«Hi»** (латиница; регистр для `Hi` не важен). Допустимы знаки препинания сразу после слова: «Привет!», «Hi,».

**Если полная сессия:**
1. Прочитать `./AGENTS.FULL.md` **от начала до конца** (все §).
2. Прочитать `./AGENTS.SETTINGS.md` и учесть все настройки с `value: true`.
3. Прочитать `./AGENTS.NODES.md`.
4. Прочитать все ноды с `AWN-LOAD: start` из реестра — они остаются в контексте до конца сессии.
5. Ноды с `AWN-LOAD: on_demand` на старте **не читать** — только при явном совпадении запроса с `AWN-TRIGGERS`.
6. Не спрашивать разрешения на чтение файлов.
7. В стартовом сообщении не перечислять прочитанные файлы и протоколы.

**Если первая реплика не подходит под правило выше — чистая сессия:**
- **Немедленно прекратить** чтение протокола AWN: **не** открывать `./AGENTS.FULL.md`.
- **Не** читать `./AGENTS.NODES.md` и ноды по протоколу AWN.
- Отвечать только по явной задаче и явно приложённым пользователем файлам.

**Важно:** только **первое** сообщение задаёт режим. Если пользователь позже напишет «Привет» или «Hi», это **не** переключает сессию на полную и **не** требует читать `AGENTS.FULL.md`.
