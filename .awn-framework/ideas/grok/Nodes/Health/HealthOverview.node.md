# Health — Обзор и Системный Подход к Здоровью

**Философия:** Здоровье — это фундамент всей экосистемы. Без него невозможно долгосрочное развитие.

**Что отслеживаем (8 ключевых сфер):**
1. Сон
2. Питание
3. Движение
4. Ментальное здоровье
5. Энергия/восстановление
6. Профилактика
7. Биохакинг
8. Регулярные чек-ины

**Мои возможности:**
- Еженедельные/ежемесячные health-review
- Корреляция между сферами (сон ↔ продуктивность и т.д.)
- Предложения по улучшению на основе твоих данных
