# Memory Palace — Долгосрочная Система Памяти

**Концепция:**  
Мы используем метод "Дворец Памяти" + Zettelkasten + Obsidian, чтобы важная информация оставалась доступной даже через годы.

**Как работает система:**

- Важные факты связываются через bidirectional links и Dataview
