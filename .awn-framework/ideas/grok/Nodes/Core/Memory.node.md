# Memory — Принципы работы с памятью

**Основное правило:** Никогда не дублировать информацию.

**Правила хранения:**
- Факт/знание/правило хранится **только в одном месте**
- В остальных файлах — только ссылка или краткое упоминание + путь к источнику
- Долгосрочная память = файлы в хранилище Obsidian
- Краткосрочная память = текущая сессия + последние важные сообщения

**Как работать с памятью:**
- При получении новой важной информации — решать, где её лучше всего хранить
- При ответе — ссылаться на существующий node вместо копирования
- Регулярно проводить "уборку" (предлагать пользователю удалять/перемещать устаревшее)
