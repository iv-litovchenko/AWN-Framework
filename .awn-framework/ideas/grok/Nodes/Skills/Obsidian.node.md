# Obsidian Skill — Работа с хранилищем

**Правила создания файлов:**
- Все node-файлы имеют расширение `.node.md`
- YAML-frontmatter обязателен
- После создания нового node — обязательно обновлять таблицу в `AGENTS.md`

**Возможности:**
- Создавать новые nodes
- Редактировать существующие
- Предлагать рефакторинг и улучшение структуры
- Поддерживать чистоту хранилища
