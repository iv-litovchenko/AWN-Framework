# Area — Nodes

> [!info] О файле
> Описывает назначение папки для контент-нод ассистента.

## Назначение

Nodes — папка для **тематических нод ассистента**: навыки, специализированные правила, конкретные инструкции по задачам. Здесь живёт всё, что расширяет возможности агента, но не описывает его идентичность (для этого — `Assistant.Ai/` корень).

## Что может здесь лежать

- Ноды навыков (как писать, как считать, как анализировать)
- Ноды для конкретных доменов (финансы, код, тексты)
- Ситуативные правила и инструкции
