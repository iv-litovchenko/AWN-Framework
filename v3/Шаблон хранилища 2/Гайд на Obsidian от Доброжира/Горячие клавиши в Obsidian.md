
| Комбинация клавиш           | Что делает                                                    |
| --------------------------- | ------------------------------------------------------------- |
| `Ctrl+P`                    | Палитра команд — то место, где можно выполнить любое действие |
| `Ctrl+O`                    | Меню быстрого перехода — поиск заметок по названию            |
| `Ctrl+Alt+O`                | Omnisearch — поиск заметок по названию **и содержимому**      |
| `Ctrl+E`                    | Режим редактирования/просмотра                                |
| `Ctrl+Alt+W`                | Режим кода                                                    |
| `Ctrl+;`                    | Добавить свойство файла (тэги)                                |
| `Ctrl+N` или `Ctrl+Shift+N` | Создать заметку                                               |
| `Ctrl+H`                    | Найти и изменить необходимые слова                            |
| `Alt+C`                     | Переместить текущий файл в другую папку                       |
| `Ctrl+F`                    | Поиск по слову внутри заметки                                 |
| `Ctrl+D`                    | Удалить абзац                                                 |
| `Alt+ЛКМ`                   | Редактировать в нескольких местах сразу                       |
| `Ctrl+W`                    | Закрыть текущую вкладку                                       |
