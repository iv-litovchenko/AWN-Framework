# mdbase examples in Tests

В `v3/Tests` добавлены 2 рабочие мини-коллекции:

1. `mdbase-demo` — задачи + люди + ссылки между записями (`link`).
2. `mdbase-habits-demo` — привычки (`enum`, `integer`, `boolean`).

Быстрый старт:

- Открой `mdbase.yaml` в каждой коллекции.
- Посмотри типы в `_types/`.
- Посмотри реальные записи в `tasks/`, `people/`, `habits/`.
- Возьми запросы из `queries.md`.
