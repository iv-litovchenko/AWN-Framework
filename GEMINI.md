# GEMINI.md

Этот файл — точка входа для Gemini-совместимых агентных сессий.

## Bootstrap (обязательно)

Перед первым ответом в каждой новой сессии ты ОБЯЗАН:

1. По **первому** сообщению пользователя определить режим: в **начале** есть приветствие вроде «Привет» или «Hi» — **полная сессия**; иначе — **чистая сессия**.

2. **Полная сессия:** прочитать `./AGENTS.md`, затем `./AGENTS.NODES.md`. На старте загружать только узлы с режимом `start` (в таблице: «при старте»). Узлы `on_demand` (в таблице: «по необходимости») на старте НЕ читать; обращаться к ним только при совпадении триггеров запроса. Не спрашивать разрешения на чтение этих файлов.

3. **Чистая сессия:** **не** читать `./AGENTS.md` и **не** читать `./AGENTS.NODES.md`; отвечать по явной задаче и по явно приложённым файлам.

4. Только после выполнения шагов по выбранному режиму формировать первый ответ.

5. Если выбрана **полная сессия** и `./AGENTS.md` не найден в текущей рабочей директории — сообщи об этом и попроси пользователя указать корректный путь.

6. В стартовом сообщении не перечисляй внешние протоколы и не предлагай читать файлы вне текущего проекта.

## Источник правил

`AGENTS.md` (в корне текущего проекта) — единый источник системных договорённостей **в полной сессии**; в чистой сессии этот файл намеренно не загружается.
