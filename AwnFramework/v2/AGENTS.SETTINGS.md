# AGENTS.SETTINGS.md

Системные настройки поведения агента для этого vault.

**Приоритет применения:**
1. Явная инструкция пользователя в текущем запросе
2. Активная настройка из этого файла
3. Умолчания из `AGENTS.md`

Если настройка не нужна — снять галочку (`[ ]`).

---

## Реестр настроек для агента

### Rendering

- [x] `links.markdown_compatible` — Использовать стандартный Markdown для ссылок и изображений вместо `[[wiki-синтаксиса]]` (совместимо с GitHub).

| Элемент        | Obsidian wiki       | Markdown                     |
| -------------- | ------------------- | ---------------------------- |
| Изображение    | `![[photo.png]]`    | `![фото](photo.png)`         |
| Ссылка на файл | `[[README]]`        | `[текст](README.md)`         |
| Якорь          | `[[README#раздел]]` | `[раздел](README.md#раздел)` |

В Obsidian: Settings → Files and links → выключить **Use [[Wikilinks]]**, формат → `Relative path to file`.

### Naming

- [x] `naming.camel_case` — При создании новых файлов и папок использовать CamelCase, если пользователь не попросил иначе

### Greeting

- [x] `session.ai_fact_on_start` — При старте новой сессии добавлять 1 короткий интересный факт из мира ИИ

### Navigation

- [x] `paths.workspace_root_relative` — Всегда указывать полный путь к файлам относительно корня хранилища (workspace), например: `Domains/Collections/Index.node.md`.

### Voice

- [ ] `voice.model_1.tts_enabled` — Разрешить TTS (Text-to-Speech) для голосового озвучивания ответов.
- [ ] `voice.model_1.stt_enabled` — Разрешить STT (Speech-to-Text) для распознавания голосового ввода.

---

## Как добавить настройку

1. Добавить строку в реестр в формате `- [x] key — описание` (или `- [ ]`, если выключено по умолчанию).
2. Использовать короткий и стабильный ключ в dot-нотации (например, `session.ai_fact_on_start`).
3. Размещать настройку в тематическом блоке (`Rendering` / `Naming` / `Greeting` / `Navigation` / `Voice`).
4. Если подходящего блока нет — создать новый тематический блок и добавить настройку в него.
5. Делать описание в 1 строку: что включает настройка и в каком контексте применяется.
6. Если агент не знает, где хранить договоренность (конфигурацию/параметр), по умолчанию зафиксировать её в этом файле.
