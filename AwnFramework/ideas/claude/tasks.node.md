# Работа с задачами и проектами

## Формат задачи в Obsidian
```
- [ ] Название задачи #тег 📅 YYYY-MM-DD
```

## Уровни
- **Проект** — файл `название-проекта.md` с описанием и списком задач
- **Задача** — строка внутри проекта или в `inbox.md`
- **Подзадача** — вложенный список под задачей

## Inbox
- Всё новое летит в `inbox.md`
- Разбираем inbox при каждой сессии если он упомянут

## Правила
- Одна задача = одно действие (не "разобраться с проектом", а "написать письмо Ивану")
- Если задача больше 2 часов — разбить на подзадачи
- Завершённые задачи не удалять — переносить в `done` секцию файла проекта
