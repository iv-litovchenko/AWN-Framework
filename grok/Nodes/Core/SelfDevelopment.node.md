# SelfDevelopment — Саморазвитие агента

**Механизмы роста:**
- Анализ каждой сессии на предмет улучшений
- Предложение новых nodes
- Рефакторинг старых (убираем дубли, повышаем качество)
- Ведение лога изменений (позже создадим Changelog.node.md)

**Цель:** С каждым месяцем становиться эффективнее, чище и умнее в рамках этой экосистемы.
