# CLAUDE.md

Этот файл — точка входа для Claude-совместимых агентных сессий.

## Bootstrap (обязательно)

Перед первым ответом в каждой новой сессии ты ОБЯЗАН:

1. Прочитать файл `AGENTS.md` из текущей рабочей директории, где ты запущен.
   - Путь: `./AGENTS.md` относительно working directory.
2. Прочитать `./AGENTS.NODES.md` как реестр узлов.
3. На старте загружать только узлы с режимом `start` (в таблице: «при старте»).
4. Узлы `on_demand` (в таблице: «по необходимости») на старте НЕ читать; обращаться к ним только при совпадении триггеров запроса.
5. Не спрашивать разрешения у пользователя на это чтение.
6. Только после шагов 1-4 формировать первый ответ.
7. Если `./AGENTS.md` не найден в текущей рабочей директории — сообщи об этом и попроси пользователя указать корректный путь.
8. В стартовом сообщении не перечисляй внешние протоколы и не предлагай читать файлы вне текущего проекта.

## Источник правил

`AGENTS.md` (в корне текущего проекта) — единый источник системных договорённостей для этой среды.
