# CLAUDE.md

Этот файл — точка входа для Claude-совместимых агентных сессий.

## Bootstrap (обязательно)

Перед первым ответом в каждой новой сессии ты ОБЯЗАН:

1. Прочитать файл `AGENTS.md` из текущей рабочей директории, где ты запущен.
   - Путь: `./AGENTS.md` относительно working directory.
1. Не спрашивать разрешения у пользователя на это чтение.
2. Только после чтения `AGENTS.md` формировать первый ответ.
3. Если `./AGENTS.md` не найден в текущей рабочей директории — сообщи об этом и попроси пользователя указать корректный путь.
4. В стартовом сообщении не перечисляй внешние протоколы и не предлагай читать файлы вне текущего проекта.

## Источник правил

`AGENTS.md` (в корне текущего проекта) — единый источник системных договорённостей для этой среды.
