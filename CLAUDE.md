# CLAUDE.md

Этот файл — точка входа для Claude-совместимых агентных сессий.

## Bootstrap (обязательно)

Перед первым ответом в каждой новой сессии ты ОБЯЗАН:

1. Автоматически прочитать файл `AGENTS.md` (путь относительный от корня текущего проекта).
2. Не спрашивать разрешения у пользователя на это чтение.
3. Только после чтения `AGENTS.md` формировать первый ответ.
4. Если файл недоступен, сразу сообщить об ошибке пути и попросить корректный путь.

## Источник правил

`AGENTS.md` — единый источник системных договоренностей для этой среды.
